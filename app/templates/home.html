INFO3180 Project 1

A Flask based application that can accept and display user profile information.

  • The user profile information is stored in a PostgreSQL database.
  • The application also has an API feature that returns JSON data for POST requests made.
  • To test the API feature, the following commands can be used to:
    1. Retrieve a list of all users
      curl -H "Content-Type: application/json" -X POST http://0.0.0.0:8080/profiles
    2. Retrieve an individual user profile
      curl -H "Content-Type: application/json" -X POST http://0.0.0.0:8080/profile/[userid]
